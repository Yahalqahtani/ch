# Attendence System

## Introduction :

### Purpose:

-   system allow normal user :
    -   create account
    -   login and logout
    -   view history for attandance
    -   make checkin and checkout
-   system allow admin user :
    -   same normal user benefits
    -   view all users
    -   update users data

### how run project:

-   open file in vscode
-   open terminal and run
    run on: http://localhost:3000/

```
npm install

npm run dev

```

-   prisma db
    run on: http://localhost:5555

```
npx prisma generate
```
