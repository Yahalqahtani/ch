# Attendence System

## Introduction :

### Purpose:

-   system allow normal user :
    -   create account
    -   login and logout
    -   view history for attandance
    -   make checkin and checkout
-   system allow admin user :
    -   same normal user benefits .
    -   view all users
    -   update users data
